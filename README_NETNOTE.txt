NETNOTE Project Copy
====================

This folder contains the full source code and built executables for NetNote (v0.1.14).

EXECUTABLES
-----------
- NetNote_Setup.exe : Installer for Windows (Recommended)
- NetNote.exe       : Portable executable

SOURCE CODE
-----------
The source code is included in this folder.
Use `npm install` to install dependencies if you wish to continue development.
Use `npm run dev` to start the development server.
Use `npm run tauri build` to recompile.

NOTES
-----
- 'node_modules' and 'target' directories were excluded to save space. Run 'npm install' to restore dependencies.
- All "Note67" branding has been replaced with "NetNote".
